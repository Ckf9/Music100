# Music100 Project Rules

## Git Repository
- The project is linked to GitHub repository: `https://github.com/Ckf9/Music100.git`.
- **Constraint**: Always commit changes and keep the Git repository clean to prevent file loss/corruption, especially since the project resides in a Google Drive stream directory.

## Core Features & Constants
- **Drake Easter Egg**: Forced background blur for all Drake tracks except "First Person Shooter".
- **Hinglish Lyrics**: `LRCParser` supports Devanagari-to-Latin transliteration using `CFStringTransform` for Hinglish lyrics syncing.
- **Lyrics View Background**: Background image opacities must be `0.59` for dark covers and `0.44` for light covers, with no bottom mask/rectangle gradients overlay.
